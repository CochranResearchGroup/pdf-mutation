## Scope

`v0.1.2` is a maintenance release for the public dogfood and reporting surface:

- `pdf-inventory` structural inventory reports and fail-on gates.
- `pdf-dogfood` policy wrapper for routine, complete, and readiness runs.
- JSONL dogfood run manifests that avoid literal probe strings.
- `pdf-dogfood-summary` manifest filters, latest-by-policy view, and health gate.
- Manual GitHub Actions dogfood health workflow using the synthetic fixture.
