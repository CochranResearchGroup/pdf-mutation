## Scope

`v0.1.3` is a maintenance release for the dogfood manifest summary surface:

- `pdf-dogfood-summary --markdown` writes pasteable Markdown summary tables.
- `pdf-dogfood-summary --output PATH` writes TSV, JSON, Markdown, or health
  output directly to a file and creates parent directories.
- `pdf-dogfood-summary --json --markdown` fails clearly instead of choosing an
  implicit output format.
- No glyph replacement behavior changed.
